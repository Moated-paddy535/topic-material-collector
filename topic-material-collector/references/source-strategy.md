# Source and Search Strategy

Read this reference when building broad, non-YouTube research or when the user needs public/commercial reuse options.

## Priority

1. Official and public archives: Internet Archive, Wikimedia Commons, Library of Congress, government archives, national libraries, museums, and topic-specific collections.
2. Educational and institutional sources: universities, government agencies, scientific organizations, cultural institutions, and official event channels.
3. Video platforms: YouTube, Vimeo, Bilibili, creator channels, documentary references, and broadcaster excerpt pages.
4. Stock and paid archives: Getty, AP Archive, Pond5, Shutterstock, Adobe Stock, Alamy, and specialist libraries.
5. Reconstruction fallback: diagrams, maps, animation, reenactment, or clearly labeled AI-generated visuals.

## Query Packs

Create 3–6 queries for each visual need:

- Literal: subject + action + place + year.
- Archival: subject + `archive footage`, `newsreel`, `raw footage`, `restored`, or repository name.
- Detail: object, uniform, device, building, map, close-up, or motion.
- Substitute: a broader event or visually equivalent action.
- Chinese: for local topics and explanatory references.
- English: usually stronger for international archives and historical footage.

Useful operators:

```text
"exact phrase" archive footage
site:archive.org "subject"
site:commons.wikimedia.org "subject"
site:loc.gov "subject"
site:youtube.com "subject" "raw footage"
```

## Selection

- `A`: direct visual match, credible source, usable resolution/motion, clean edit point.
- `B`: useful approximation requiring crop, context, captions, grade, or compositing.
- `C`: research/reference only or weak visual match.

Prefer several high-fit sources over dozens of loosely related results. Keep a backup for fragile links or licensed sources.

## Verification

- Open the source page and verify title, creator, date, and license metadata.
- For videos, confirm that the stated timestamp contains the requested picture, not only related narration.
- Record login, regional, payment, and licensing constraints.
- Never label an asset public domain solely because a repost title says so.
